# Portfolio Analyser

Portfolio Analyser is a Streamlit application that lets you upload a holdings CSV, review and enrich the portfolio using Yahoo Finance metadata, analyse asset and regional allocation, calculate trailing portfolio performance over 1, 3 and 5 years, and generate a client-ready PowerPoint presentation. The app accepts a CSV containing a holding identifier such as a ticker, ISIN or SEDOL, a description, and a GBP value, then classifies holdings where possible, highlights anything unresolved, and produces portfolio insights through charts and tables. Once the review is complete, the user enters the client’s name and the tool generates a three-slide `.pptx` file containing a title slide called **Portfolio Review for [Client Name]**, a performance table slide showing 1-year, 3-year and 5-year portfolio returns, and a final **Thank you for your time** slide. To run the project, install the dependencies with `pip install -r requirements.txt` and then start the app with `streamlit run app.py`. The expected CSV should contain columns equivalent to identifier, description and GBP value, for example: `ticker,description,gbp_value` followed by rows such as `AAPL,Apple Inc,12500` or `CASHGBP,Cash,2500`. The main Python dependencies are `streamlit`, `pandas`, `plotly`, `yfinance`, and `python-pptx`, with `python-pptx` being used so the presentation can be created locally without any Google Slides API access.
